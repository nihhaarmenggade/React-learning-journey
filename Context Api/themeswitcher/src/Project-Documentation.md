# 🌗 Theme Switcher

A modern React application that demonstrates light and dark theme switching using the React Context API. The project features a premium UI card with a custom theme toggle, allowing users to seamlessly switch between themes across the application.

## 📝 Project Summary

This project was built to understand global state management in React using the Context API. It showcases how a theme can be managed efficiently across components without prop drilling. The application includes a responsive UI, custom-designed theme switch, and a premium card layout styled with Tailwind CSS.

## 🖥️ How It Works

- A global theme state is created using the React Context API.
- The `ThemeProvider` wraps the application and provides the current theme.
- The custom `ThemeBtn` component updates the global theme state.
- All components automatically update their styles based on the selected theme.
- Tailwind CSS classes are applied dynamically to create a smooth light/dark mode experience.
