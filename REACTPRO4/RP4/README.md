💱 Currency Converter (React + Tailwind CSS)

A modern Currency Converter web application built with React and Tailwind CSS, featuring a custom React hook (useCurrencyInfo) for handling API logic and a reusable JSX component (InputBox) for inputs and currency selection. The app fetches real-time exchange rates and provides a clean, responsive UI.

🌐 Check the live project :
https://react-learning-journey-apce-pnttgctq8.vercel.app/


## 🚀 Features

🔁 Convert currencies in real time

🌍 Supports all major world currencies

🎨 Modern UI with Tailwind CSS & gradients

🧠 Custom React Hook for API handling

📱 Fully responsive (desktop & mobile)

⚡ Fast development using Vite


## 🛠️ Tech Stack

This project uses the following free public currency API:

https://currency-api.pages.dev/v1/currencies/{currency}.json

Example:https://currency-api.pages.dev/v1/currencies/usd.json


## 🎯 Key Learnings from This Project

Creating and using custom React hooks

Handling async API calls with useEffect

Controlled components in React

Managing UI bugs caused by CSS vs logic
